# registration-form
registration form for social media (done as part of the training)

---

## Functional Requirements

1. **User Input**: The user must be able to enter a name, email, and password.
2. **Mandatory Fields**: The system must check that all required fields are filled in.
3. **Password Match**: The system must check that the password and the password confirmation match.
4. **Submission Logic**: If all data is valid, the form must simulate submission and show a success message.

---

## Non-Functional Requirements

1. **Semantic Markup**: The page must be implemented using semantic HTML markup.
2. **Style Separation**: All styles must be placed in an external CSS file.
3. **Cross-browser Compatibility**: The page must display correctly in modern web browsers.
4. **Error Messaging**: Validation messages must be clear and understandable for the user.
